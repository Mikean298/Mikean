Học CSS: Lộ trình chi tiết cho người mới bắt đầu

1. LỊCH SỬ CỦA CSS

CSS được phát triển bởi World Wide Web Consortium (W3C). CSS ra đời với mục đích đơn giản là định hình cho website thông qua màu sắc, font chữ, bố cục... giúp tách biệt nội dung (HTML) và hình thức hiển thị.

Cha đẻ của CSS

  • CSS1 (1996): Những tính năng cơ bản như màu sắc, font chữ.
  • CSS2 (1998): Bổ sung bố cục phức tạp và hỗ trợ đa phương tiện.
  • CSS3 (2011): Bước tiến lớn với Flexbox, Grid, Animation.
  • CSS4: Đang phát triển để tối ưu trải nghiệm người dùng.

2. KHÁI NIỆM MẪU ĐỊNH DẠNG CSS

CSS là định dạng độc lập với chuẩn HTML. Ví dụ về cách định dạng:

  • Mẫu 1: Thiết lập chữ đỏ cho thẻ <h1>.
  • Mẫu 2: Khung viền xanh 2px cho <h1>.
  • Mẫu 3: Thụt lề 15px cho thẻ <p>.

Mẫu định dạng CSS

3. TÌM HIỂU VỀ CSS

CSS là công cụ trang trí, giúp HTML trở nên sinh động. Các loại CSS thông dụng:

  • CSS Background (Hình nền)
  • CSS Text & Font (Chữ viết)
  • CSS Box Model (Padding, Margin, Border)
  • CSS Table & List

4. CẤU TRÚC VÀ CÁCH SỬ DỤNG

Cấu trúc tổng quát:

selector {
  property: value;
}

3 Cách thiết lập:

  • CSS trong (Internal): Đặt trong thẻ <style> ở head.
  • CSS ngoài (External): Lưu file .css riêng và dùng thẻ <link>.
  • CSS nội tuyến (Inline): Viết trực tiếp vào thuộc tính style="..." của thẻ HTML.

5. ƯU VÀ NHƯỢC ĐIỂM

Ưu điểm: Tiết kiệm thời gian, tải trang nhanh, dễ bảo trì, hỗ trợ đa thiết bị (Responsive).

Nhược điểm: Hoạt động khác nhau trên mỗi trình duyệt, khó cho người mới, rủi ro về định dạng nếu file lỗi.

6. VAI TRÒ VÀ Ý NGHĨA

CSS giúp tách biệt nội dung và định dạng, tăng tính chuyên nghiệp và cho phép chỉnh sửa đồng bộ hàng loạt trang web chỉ bằng một thay đổi nhỏ.
