1. LỊCH SỬ CSS

CSS được phát triển bởi W3C, là tổ chức chuyên xây dựng các tiêu chuẩn web.

CSS ra đời nhằm định hình website thông qua màu sắc, font chữ, bố cục, hình ảnh…

CSS giúp tách nội dung HTML và giao diện, giúp web chuyên nghiệp hơn.

Các phiên bản:

  • CSS1 (1996): màu sắc, font, căn lề
  • CSS2 (1998): bố cục phức tạp
  • CSS3 (2011): animation, flexbox, grid
  • CSS4: đang phát triển

2. KHÁI NIỆM CSS

CSS là ngôn ngữ định dạng giao diện website.

CSS giúp trang web trở nên sinh động và thân thiện với người dùng.

  • Background
  • Text
  • Font
  • Table
  • Margin, padding

3. CẤU TRÚC CSS

CSS gồm 2 phần:

  • Selector
  • Declaration
h1 {
  color: red;
  font-size: 20px;
}

4. CÁCH SỬ DỤNG CSS

  • Internal CSS

  • Inline CSS

5. ƯU VÀ NHƯỢC ĐIỂM

Ưu điểm:

  • Tiết kiệm thời gian
  • Dùng lại nhiều lần
  • Web đẹp hơn

Nhược điểm:

  • Khác trình duyệt
  • Khó cho người mới
  • Có thể lỗi
