📜 1. Lịch sử CSS

CSS được phát triển bởi W3C nhằm tách nội dung và giao diện web.

CSS1 (1996): Định dạng cơ bản
CSS2 (1998): Bố cục nâng cao
CSS3 (2011): Animation, Flexbox, Grid
CSS4: Đang phát triển

💡 2. Khái niệm CSS

CSS là ngôn ngữ dùng để định dạng giao diện website.

  • Màu sắc
  • Font chữ
  • Bố cục
  • Khoảng cách

🧱 3. Cấu trúc CSS

Gồm 2 phần:

  • Selector
  • Declaration
h1 {
  color: red;
  font-size: 20px;
}
