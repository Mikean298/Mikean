📜 Lịch sử CSS

CSS được phát triển bởi W3C nhằm tách nội dung và giao diện.
CSS1 (1996): Cơ bản (màu, font, căn lề)
CSS2 (1998): Bố cục phức tạp hơn
CSS3 (2011): Animation, Flexbox, Grid
CSS4: Đang phát triển

💡 Khái niệm CSS

CSS là ngôn ngữ dùng để định dạng giao diện website.

  • Màu sắc
  • Font chữ
  • Bố cục
  • Khoảng cách

🧱 Cấu trúc CSS

Selector: chọn phần tử HTML
Declaration: thuộc tính + giá trị 
h1 {
  color: red;
  font-size: 20px;
}

⚙️ Cách sử dụng CSS

Internal CSS (trong thẻ <style>)
External CSS (file .css)
Inline CSS (trực tiếp trong thẻ)

⚖️ Ưu và Nhược điểm

✅ Ưu điểm

  • Tiết kiệm thời gian
  • Tái sử dụng code
  • Web đẹp hơn

❌ Nhược điểm

  • Khác nhau giữa trình duyệt
  • Khó với người mới
  • Có thể lỗi nếu file CSS hỏng
